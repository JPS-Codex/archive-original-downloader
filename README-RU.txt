Назначение

Перед скачиванием с сайта ЦГА Московской области обязательно войдите в личный кабинет:
https://arch.mosreg.ru/login

Если учётной записи ещё нет, зарегистрируйтесь:
https://arch.mosreg.ru/registration

Без входа в личный кабинет сайт может не отдавать изображения или может открыть только часть данных.

Установка в Chrome

1. Распакуйте ZIP-архив расширения в отдельную папку.
2. Откройте страницу расширений: chrome://extensions/
3. Включите режим разработчика.
4. Нажмите "Загрузить распакованное расширение".
5. Выберите папку с распакованным расширением.

Установка в Opera

1. Распакуйте ZIP-архив расширения в отдельную папку.
2. Откройте страницу расширений: opera://extensions/
3. Включите режим разработчика.
4. Нажмите "Загрузить распакованное расширение".
5. Выберите папку с распакованным расширением.
6. Для Яндекс.Архива обязательно включите у расширения галочку "Разрешить доступ к результатам на странице поиска". Если её не включить, кнопка скачивания на странице архива может не появиться.

Как пользоваться

1. Откройте страницу со сканом документа.
2. Дождитесь полной загрузки изображения.
3. Для одного скана нажмите "Скачать скан".
4. Для пакетного скачивания нажмите "Скачать всё дело".
5. При необходимости задайте диапазон "С" и "По".
6. Выберите "Одним ZIPом" или "Отдельными файлами JPG".
7. Нажмите "Да", чтобы начать скачивание.

Во время пакетного скачивания можно прервать процесс. Для ЦГА Московской области страница может автоматически переходить по номерам изображений; это нормальная часть работы пакетного режима.

Примеры имён файлов

- Яндекс.Архив: ЦГА_Москвы_[ЯА]_203-780-880-0005.jpg
- ЦГА Москвы: ЦГА_Москвы_203-745-630-0001.jpg
- ЦГА Московской области: ЦГА_МосОбл_2510-1-850-#обл(цв).jpg

Что сохраняется в ZIP

В ZIP помещаются JPG-файлы сканов и служебный файл info.txt. Если при пакетном скачивании часть страниц не удалось подготовить, дополнительно создаётся errors.txt.

Если кнопка не появилась

- Обновите страницу архива.
- Проверьте, что расширение включено.
- Проверьте, что сайт входит в список поддерживаемых.
- В Opera для Яндекс.Архива проверьте галочку "Разрешить доступ к результатам на странице поиска".
- Для ЦГА Московской области проверьте, что выполнен вход в личный кабинет.
