# Changelog

## 1.8.1-beta

- Added support for the Moscow Region archive image viewer at `arch.mosreg.ru`.
- Added bulk downloading for the Moscow Region archive using page ranges.
- Added ZIP packaging for bulk downloads.
- Added meaningful file names for Moscow Region archive scans.
- Added `[ЯА]` marker to Yandex Archive file names.
- Added support for multiple CGAMOS sections.
- Added range downloads, pause, cancel, retry failed pages and skip already downloaded pages.
- Added Russian user documentation.
